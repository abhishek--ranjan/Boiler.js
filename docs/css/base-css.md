# Base CSS

- No Bootstrap, Tailwind, or external CSS frameworks
- CSS is layered: base, layout, component
- Uses CSS Grid and Flexbox for layout
- CSS variables for theming and overrides
- Users may override styles via variables or custom classes

## Allowed
- CSS variables
- Grid and Flexbox
- Scoped component styles

## Forbidden
- Global style overrides
- Framework-level !important
